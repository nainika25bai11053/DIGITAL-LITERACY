##  Task 5: Cyber Crime Awareness

### 🔹 Case Study: UPI Payment Fraud

* Fraudsters trick users into entering UPI PIN
* Common methods:

  * Fake calls/messages
  * Payment request scams
* **Impacts:**

  * Financial loss
  * Mental stress

### 🔹 Prevention Tips:

* Never share OTP or UPI PIN
* Avoid unknown links
* Verify payment requests
